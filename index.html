如何进入一个处于运行时的docker容器:

主要有三种方式：

  • (1)使用docker命令attach

    • docker attach containerid

  • (2)使用docker命令exec

    • docker exec containerid

关于在我们执行docker　run 后面一般都会加-it参数：

-i:参数是不会分配终端的，就像但是可以执行命令并且看输出结果．   
-t:参数是分配终端，但是看不到命令执行的输出结果．  
所以一般都是这两个参数的结合．

注意：

　　上面两种方式的区别是很大的，docker attact是到一个已经运行的容器的stdin，然后进行命令执行的动作， 　　当执行exit的时候则该容器也会随之退出, 而使用docker exec是不会的，此时相当于又启了一个docker子进程．

  • (3)使用sshd或是第三方工具
    使用shh则需要docker安装sshd服务．
